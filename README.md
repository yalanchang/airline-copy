# 華航動物方城市2主題彩繪機 網站

##  預覽

- 飛機與標題同步浮動動畫
- 雲朵從右至左持續飄動
- 滑鼠互動裝飾圓圈
- 滾動視差效果
- 曲線遮罩過渡

## 技術棧

- **框架**: Next.js 14 (App Router)
- **語言**: TypeScript
- **樣式**: Tailwind CSS
- **動畫**: CSS Keyframes + JavaScript

## 功能特色

| 功能 | 說明 |
|------|------|
| 飛機浮動 | 飛機與標題一起上下微微浮動 |
| 雲朵飄動 | 雲朵從右邊持續飄向左邊 |
| 圓圈互動 | 裝飾圓圈跟隨滑鼠移動 + 滾動視差 |
| 滾動視差 | 不同元素以不同速度移動 |
| 曲線遮罩 | SVG 曲線過渡到內容區 |
